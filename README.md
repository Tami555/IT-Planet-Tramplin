# Трамплин — Карьерная платформа

> Платформа для централизованного взаимодействия студентов, выпускников и IT-работодателей. Проект-победитель конкурса «Прикладное программирование if...else» международной олимпиады «IT-Планета 2026».

## 🎯 О проекте

**Трамплин** — это интерактивная карьерная экосистема, где соискатели находят стажировки, вакансии и менторские программы, а работодатели публикуют возможности и находят талантливых кандидатов.

### Реализованный функционал

- **Интерактивная карта** с кластеризацией маркеров и группировкой нескольких вакансий на одном адресе
- **Поиск и фильтрация** по навыкам, зарплате, формату, городу (более 10 параметров)
- **Личный кабинет соискателя**: профиль, отклики, избранное, профессиональные контакты, настройки приватности
- **Личный кабинет работодателя**: управление вакансиями, просмотр откликов, верификация компании
- **Панель куратора**: модерация контента, верификация компаний, управление пользователями и тегами
- **Профессиональные контакты** с настройками приватности (PUBLIC/CONTACTS/PRIVATE)

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- PostgreSQL 14+
- npm 9+ или yarn 1.22+

### Бэкенд

```bash
cd tramplin--server
cp .env.example .env          # Настройте DATABASE_URL и JWT_SECRET
npm install
npm run prisma:migrate        # Создание таблиц
npm run prisma:seed           # Заполнение начальными данными
npm run start:dev             # Запуск на http://localhost:3000
```

### Фронтенд

```bash
cd frontend
cp .env.example .env          # Укажите REACT_APP_YANDEX_MAPS_API_KEY
npm install
npm start                     # Запуск на http://localhost:5000
```

## 🔑 Авторские решения (обоснование)

| Решение | Реализация | Обоснование |
|---------|------------|-------------|
| **Верификация компаний** | Корпоративный email (авто) + ИНН (куратор) | Баланс удобства и надёжности |
| **Модерация контента** | Очередь с комментариями при отклонении | Не блокирует работу, помогает исправлять ошибки |
| **Приватность профиля** | 3 уровня: PUBLIC / CONTACTS / PRIVATE | Гибкость для разных сценариев использования |
| **Карточка на карте** | Название, компания, логотип, тип, зарплата, 3 тега | Минимум для быстрой оценки вакансии |
| **Профессиональные контакты** | Двустороннее подтверждение (как в соцсетях) | Защита от спама, добровольный нетворкинг |
| **Доступ к соискателям** | Работодатель видит только откликнувшихся | Защита от несанкционированного просмотра |
