# Трамплин - Карьерная платформа (Фронтенд 🎨)

Фронтенд-часть платформы для централизованного взаимодействия студентов, выпускников и IT-работодателей.

## 📋 О проекте

**Трамплин** — это интерактивная карьерная платформа, где соискатели могут находить стажировки, вакансии, менторские программы и карьерные мероприятия, а работодатели — публиковать возможности и находить талантливых кандидатов.

### Основные возможности

#### Для всех пользователей:
- Просмотр вакансий и мероприятий на интерактивной карте (Яндекс.Карты)
- Поиск и фильтрация возможностей по навыкам, зарплате, формату работы, городу
- Просмотр детальной информации о возможности
- Добавление возможностей в избранное (сохраняется в localStorage для неавторизованных)

#### Для соискателей (студенты/выпускники):
- Личный кабинет с профилем (ФИО, вуз, курс, навыки, портфолио)
- Отклик на возможности с отслеживанием статуса
- Избранное для сохранения интересных предложений
- Система профессиональных контактов (друзья/нетворкинг)
- Настройки приватности профиля (публичный/только контакты/только я)
- История откликов с фильтрацией по статусам

#### Для работодателей:
- Профиль компании с информацией, логотипом и фото офиса
- Создание и управление карточками возможностей (стажировки, вакансии, менторство, мероприятия)
- Просмотр откликов соискателей с возможностью изменения статуса
- Верификация компании (через заявку куратору)
- Справочник тегов для создания возможностей

#### Для кураторов/администраторов:
- Панель управления с общей статистикой платформы
- Управление пользователями (просмотр, блокировка)
- Верификация компаний (рассмотрение заявок)
- Модерация возможностей (одобрение/отклонение)
- Управление системными тегами (создание, удаление)

### Архитектурные решения

```
src/
├── api/              # API-клиент и сервисы для работы с бэкендом
│   ├── config/       # Настройки (endpoints, apiClient)
│   ├── services/     # Сервисы для каждой сущности (User, Opportunity, Applicant, Employer)
│   └── utils/        # Утилиты (обработка ошибок, работа с cookies)
├── components/       # Переиспользуемые UI-компоненты

│   ├── YandexMap/    # Компонент карты с кластеризацией маркеров
│   ├── OpportunityCard/  # Карточка возможности
│   ├── Filters/      # Панель фильтрации
│   └── ...
├── contexts/         # React Context (AuthContext, FavoritesContext)
├── hooks/            # Кастомные хуки (useFetch, useValidation, useFavorites)
├── pages/            # Страницы приложения
│   ├── MainPage/     # Главная страница (карта + список)
│   ├── LoginPage/    # Авторизация
│   ├── RegisterPage/ # Регистрация
│   ├── ApplicantProfilePage/  # Профиль соискателя
│   ├── EmployerProfilePage/   # Профиль работодателя
│   ├── CuratorProfilePage/    # Панель куратора
│   ├── ContactsPage/ # Профессиональные контакты
│   └── ...
├── utils/            # Вспомогательные функции (валидация, форматирование)
└── data/             # Моковые данные (для fallback)

```

### Ключевые особенности реализации

1. **Интерактивная карта**
   - Группировка маркеров при одинаковых координатах (кластеризация)
   - Отображение нескольких возможностей на одном адресе со скроллом в балуне
   - Разные цвета маркеров для статусов (избранное, откликнулся)

2. **Система фильтрации**
   - Реальная фильтрация на бэкенде с отправкой параметров запроса
   - Debounce для поиска (300 мс)
   - Фильтрация по навыкам, зарплате, формату, типу, городу

3. **Безопасность**
   - JWT-токены хранятся в cookies
   - Интерсепторы Axios для автоматической подстановки токена
   - Разграничение прав доступа на уровне маршрутов

4. **Профессиональные контакты**
   - Поиск соискателей с фильтрацией по навыкам, вузу, городу
   - Отправка/принятие/отклонение заявок в друзья
   - Настройки приватности профиля

## 🚀 Установка и запуск

### Требования
- Node.js 18+ 
- npm 9+ или yarn 1.22+

### Установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd tramplin-frontend

# Установка зависимостей
npm install

# Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env, указав API ключ Яндекс.Карт
```

### Запуск

```bash
# Режим разработки (порт 5000)
npm start

# Сборка для production
npm run build

# Просмотр production-сборки локально
npm run preview
```

## 🗄️ API интеграция

Фронтенд взаимодействует с бэкендом через REST API. Основные эндпоинты:

| Сущность | Эндпоинты |
|----------|-----------|
| **Auth** | `/auth/login`, `/auth/register`, `/auth/refresh`, `/auth/logout` |
| **Opportunities** | `GET /opportunities`, `GET /opportunities/:id`, `POST /opportunities/:id/apply`, `POST/DELETE /opportunities/:id/favorite` |
| **Applicants** | `GET /applicants/me`, `GET /applicants/me/applications`, `GET /applicants` (поиск) |
| **Employers** | `GET /employers/me`, `GET /employers`, `GET /employers/:id`, `PUT /employers/me` |
| **Tags** | `GET /tags`, `POST /tags`, `DELETE /tags/:id` |

Все защищенные эндпоинты требуют Bearer-токен, который автоматически подставляется интерсептором Axios.
